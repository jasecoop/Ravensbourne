  • Bringing the Olympics to the urban community

  • Beat the Feat promotes London 2012. The Beat the Feat concept uses the publicity of Olympic stars to set ‘targets’ within the urban community enabling the public to challenge the athletes. Beat the Feat uses web and smartphone applications to allow users open location access to the service wherever they are including challenge locations.

    Challenges are set using printed stickers, pink for the athletes and blue for the public, using these allow differentiation between the star and challengers. Users will upload their attempts, which will be sorted by location. User voting allows, people with the most popular and creative feats to win prizes including tickets to 2012 games or an audience with the athlete they beat.
