  • Designing Value

  • A multi functional domestic tool with interchangeable heads that allows for multiple uses within different environments.

    This project focused on the use of home appliances - specifically tools, in domestic environments. It explored the value in which different users place in the same object, how this value might be perceived by the user and how the designer contributes to this.

    The multi- functional object starts to blur the boundaries that have previously separated certain objects through their function or environment of use. This new type of product is aimed at amateur users and can be described as a domestic tool, which is placed within in the home setting.
